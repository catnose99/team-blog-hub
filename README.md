# HiTTO Product Team Blog Hub

https://product-blog-hub.hitto.co.jp/

- メンバー追加する場合は、`./members.ts` に追加してPRをお願いします。
- PRごとにデプロイされ確認できます。
- masterにマージされるとProductionにデプロイされます。
- 記事は6時間おきに収集・反映されます。

## Development

```bash
$ yarn install
$ yarn build
$ yarn dev
```
